你需要記住的 6 個名詞 Part 4/8

1. HTML（結構）

網頁的「骨架」，用來告訴電腦「內容是什麼」

2. Tag（標籤）

用尖括號包起來的東西，如 <h1>、<p>

3. Element（元素）

完整的標籤組合：開始標籤 + 內容 + 結束標籤

4. Attribute（屬性）

標籤的額外設定，如 class="title"

5. Class（分類）

給元素取「名字」的屬性，像是門牌號碼

6. 瀏覽器渲染

瀏覽器讀取 HTML 並「畫」在螢幕上的過程

# HTML 簡報製作 Skill

## 這是什麼？
這是一套用於製作 HTML 簡報的工具組，包含模板和 Prompt。

## 使用方式
1. 複製 template.html 作為新簡報的基礎
2. 使用 prompts/ 資料夾中的 Prompt 生成內容
3. 把生成的內容填入模板的【】標記處

## 模板結構說明
- 封面頁：【簡報標題】【副標題】【講者姓名】
- 內容頁：【章節標題】【重點1-5】
- 結尾頁：【感謝文字】【聯絡資訊】

## Prompt 使用順序
1. 先用「標題生成器」想標題
2. 用「重點條列器」整理每頁重點
3. 用「開場白生成器」寫開場
4. 用「結語生成器」寫結尾

## 注意事項
- 每頁重點不超過 5 個
- 每個重點不超過 20 字
- 配色請遵循模板的預設值
