## 📖 課程簡介

這是一門為期兩天的實戰工作坊，教你如何使用 AI 工具協作製作 HTML 簡報，並將成品部署到 GitHub Pages 上線。

- **時長**：2 天（每天 6 小時）
- **成果**：完整 HTML 簡報 + 上線網站
- **對象**：零基礎也能學會
